# TsaoNyeeMaa SimTrade Console

**TsaoNyeeMaa SimTrade Console** 是一个面向**对外经济贸易大学·世格软件开发的 SimTrade 外贸实习平台 V6.0**的单文件 HTML 辅助工具。它试图把进出口预算、交易角色切换、邮件广告文本、业务单据内容生成和本地存档整合到一个现代化交互页面里。

> 本项目是个人学习与课程辅助用途的原型工具，不是对外经济贸易大学或世格软件的官方产品。

每次交易都在亏钱？

每次预算表都算到注意力涣散？

每次填表都要来回检查好几次？

TsaoNyeeMaa SimTrade Console让你在脑袋空空的时候，也能稳定做完每一单。

## 功能概览

- 现代化的单页交互界面
- 完整的进出口预算计算
- 一键生成 SimTrade 业务单据内容
- 自动生成邮件与广告文本
- 支持出口商、进口商、工厂、银行等参与者便捷切换
- 支持多人互做场景
- 完善的本地存档导入 / 导出机制
- 完整的贸易术语/支付方式/保险条款/结算货币支持
- 支持一单多商品
- 内置国内港口数据
- 内置 11 种代表性商品
- 支持黑暗主题

## 使用本工具

省流：下载网页.html文件 -> 打开即用！

（如果你想一键粘贴报表）：下载pastebill.htm -> 安装注册表 -> IE右键菜单中就有了

### 1. 下载网页文件

下载本项目中的 HTML 文件，例如：

- `Tsaonyeema-SimTrade-Console_6.1.3.html`

本工具为单文件前端页面，无需安装服务器、Node.js 或其他运行环境。

### 2. 使用现代浏览器打开

打开后即可在本地使用。所有计算、表单填写和单据内容生成均在浏览器中完成。

### 3. 按工作流填写信息

页面分为三个主要步骤：

1. **必要信息**  
   填写进出口商、进出口地银行、工厂、启运港、目的港和运费等基础资料。

2. **交易设定**  
   确认交易身份、结算货币、贸易术语、支付方式、保险、报检、商品清单、价格和利润分配等内容。

3. **生成与导出**  
   生成邮件、广告、合同、预算表和业务单据 XML 内容。

### 4. 导出存档

建议在完成一次交易配置后及时导出 JSON 存档。

浏览器刷新、关闭页面或误操作可能导致当前填写内容丢失。导出的存档可在下次打开页面后重新导入。

## 可选：安装 SimTrade 页面粘贴脚本

如果需要将本工具生成的 XML 内容一键粘贴进 SimTrade 页面，可配合额外的 `pastebill.htm` 脚本使用。

> 该脚本主要用于 Internet Explorer / IE 模式下的右键菜单粘贴流程。  
> 如果你只是使用本工具进行计算、生成文本或手动复制内容，则不需要安装此脚本。

### 1. 下载 `pastebill.htm`

将 `pastebill.htm` 保存到本地固定目录，例如：

```text
C:\IEScripts\pastebill.htm
```

请不要随意移动该文件。移动后，已导入的右键菜单路径会失效，需要重新修改并导入注册表文件。

### 2. 创建注册表文件

修改注册表文件时你需要知道自己在做什么。

在`pastebill.htm`同目录下新建文件：

```text
install_paste.reg
```

使用记事本打开，并写入以下内容：

```reg
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\MenuExt\PASTE]
@="C:\\IEScripts\\pastebill.htm"
"Contexts"=dword:00000001
```

如果你把 `pastebill.htm` 放在其他目录，请将上方路径改为实际路径。

注意：注册表路径中的反斜杠需要写成双反斜杠，例如：

```text
D:\\Tools\\pastebill.htm
```

### 3. 导入注册表

双击 `install_paste.reg`，确认导入注册表。

导入后，在支持 Internet Explorer 右键菜单扩展的环境中，应能看到 `PASTE` 右键菜单项。

注册表操作会修改当前 Windows 用户下的 Internet Explorer 右键菜单配置。导入前请确认脚本路径正确。

### 4. 使用方式

在本工具中生成并复制单据 XML 内容后，进入 SimTrade 对应单据页面，使用 `PASTE` 粘贴脚本将内容填入页面。

> 注：SimTrade 的页面大多仍依赖 IE 环境。主程序建议用现代浏览器打开；粘贴脚本相关流程可能需要额外配置 IE 右键菜单。

## 已知限制

- 合同目前仅支持按出口商侧发出。
- 手机端布局暂未作为主要使用场景。
- IE 不适合作为此程序运行环境。
- 单据粘贴后建议进行检查，[信用证]已知固定有两处错误。
- 单据生成逻辑面向 SimTrade 外贸实习流程，不保证适用于真实外贸业务。

## 为什么做这个小程序

我没有系统的程序设计经验。这个项目一开始只是一个 Excel 表，后来正好赶上 Gemini 和 GPT 的前端设计能力大幅提升，于是一路把它改成了现在的单文件 HTML 工具。

某种意义上，这是我过了一把前端设计瘾的作品。如果它能帮你少算错几项预算表，少亏几笔，少来回翻几次页面，我会很荣幸。

## 未来规划

目前商品和国外港口还是需要靠用户手填，期望以后能内置完整数据。
信用证的内容有两处错误，等待修改。
可能会做IE能打开的精简版本。

如果有人愿意继续做下去，我更希望它未来变成一个完整的 SimTrade 学习引导工具：不仅能算表、填单，还能和 **对外经济贸易大学·世格软件开发的 SimTrade 外贸实习平台 V6.0** 流程结合，形成更清晰的新手引导、检查机制和错误提示。
